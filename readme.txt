Параметры запуска игры:
 - Окно фиксированного размера
 - Разрешение 1600х900

Автоматизирован следующий алгоритм:
 - выход корабля из дока и возврат его в док 
Оформлено в виде бесконечного цикла.

Для корректной работы предполагается что запуск скрипта начинается когда корабль находится в доке.

В проекте прокомментировал только сам алгоритм в процедуре main, т.к. вспомогательные процедуры были
взяты из обучающих видео.
